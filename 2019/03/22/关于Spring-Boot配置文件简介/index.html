1. YAML语言

1.1 简介

YAML是”YAML Ain’t a Markup Language”（YAML不是一种标记语言）的递归缩写，文件名以.yml结尾。

1.2 基本语法

  1. key: value表示一对键值对（冒号后面必须要有一个空格）。
  2. key和value大小写敏感。
  3. 使用空格缩进，不允许使用Tab键缩进，缩进表示层级关系，缩进的空格数量无要求（一般空两格），但同一层级要对齐。
  4. 使用#进行注释，从这个字符一直到行尾都会被解析器忽略。
