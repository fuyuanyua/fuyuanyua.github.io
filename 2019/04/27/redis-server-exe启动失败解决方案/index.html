redis-server.exe启动失败解决方案

发表于 2019-04-27 | 更新于 2019-04-28

今天我下载了Win版的Redis，解压缩后，双击redis-server.exe，屏幕一闪而过，启动失败，查阅网上资料后找到解决办法。

  • win+R打开cmd命令。
  • 进入F盘：f:
  • cd到Redis包目录下：cd F:\redis\Redis-x64-3.2.100（在当前目录的地址栏直接输入cmd，可直接进入，更方便）
  • 输入：redis-server.exe redis.windows.conf
  • 启动成功
